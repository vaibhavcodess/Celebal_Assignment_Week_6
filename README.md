# Week 6 - Apache Spark DataFrame Operations

This repository contains my Week 6 assignment completed during the Celebal Technologies Data Engineering Internship.

The assignment focuses on working with Apache Spark DataFrames using PySpark. It covers loading data from a CSV file, exploring the dataset, applying transformations and filters, handling missing values, understanding the difference between CSV and Parquet formats, and building a simple data processing pipeline.

## Topics Covered

* Spark Session Creation
* Reading CSV Files
* Schema Inspection
* DataFrame Transformations
* DataFrame Actions
* Filtering and Selection
* Handling Null Values
* CSV vs Parquet
* Data Processing Pipeline
* Spark Architecture Concepts

## Files Included

* `Spark_DataFrame_Operations.ipynb` – Main assignment notebook
* `README.md` – Project overview
* `Ques_Answers.ipynb` - Question and Answers

## Tools Used

* Python
* PySpark
* Apache Spark
* Jupyter Notebook

## Learning Outcome

This assignment helped me understand how Spark processes large datasets using DataFrames and how different transformations and actions are used in real-world data processing workflows.
